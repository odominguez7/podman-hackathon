# Podman Hackathon

Built at the Podman Hackathon.

## Stack
- Podman Desktop + AI Lab Extension
- RamaLama (local model inference)
- Claude Code (AI-assisted development)
- MCP (Model Context Protocol)

## Setup
```bash
# Pull a local model
ramalama pull granite-code

# Run inference
ramalama --nocontainer run granite-code "hello"
```
